## infinite-scroll

> A component project

### 参与开发

> 项目构建说明请参考：[打包构建](https://gitlab.gridsum.com/frontend/gs-ui/elf/wikis/home)

> 组件文档编写规范：[组件文档约定](http://gs-ui.gridsum.com/#/zh-CN/component/docs-guide)

* 下载安装依赖

```shell
npm install
```

* 启动开发环境

```shell
npm start
```

浏览器打开 http://localhost:10001 即可看到效果

* 构建打包

```shell
npm run build
```
